
## CarePoint: Dirty Clinic Patient Dataset

### Overview
This dataset contains synthetic clinic patient records intentionally designed with data quality issues for teaching data cleaning and preparation.

**Important:** This dataset is fictional and contains no real patient information.

---

## Variables

| Column Name | Description | Expected Data Type |
|------------|-------------|-------------------|
| Patient_ID | Unique identifier assigned to each patient | Text |
| First_Name | Patient's first name | Text |
| Last_Name | Patient's last name | Text |
| Gender | Patient gender | Categorical |
| Age | Patient age in years | Numeric |
| Date_of_Birth | Patient date of birth | Date |
| Phone_Number | Contact phone number | Text |
| Email | Patient email address | Text |
| Address | Residential address | Text |
| City | City of residence | Text |
| State | State/Province of residence | Text |
| Blood_Group | Patient blood group | Categorical |
| Weight | Patient weight | Numeric |
| Height | Patient height | Numeric |
| Diagnosis | Primary diagnosis recorded during visit | Categorical |
| Visit_Date | Date of clinic visit | Date |
| Doctor_Assigned | Attending physician | Text |
| Payment_Amount | Amount paid for consultation/treatment | Numeric |
| Payment_Status | Payment status | Categorical |

---
## Suggested Analyses After Cleaning

Students can perform:

- Patient demographic analysis
- Gender distribution
- Diagnosis frequency analysis
- Monthly patient visits
- Revenue analysis
- Doctor workload analysis
- Geographic patient distribution
