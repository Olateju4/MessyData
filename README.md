# MessyData
A messy dataset repository for teaching data cleaning and preparation to data analytics students.

## Purpose
The dataset helps data analytics students practice:
- Handling missing values
- Removing duplicates
- Standardizing date formats
- Cleaning phone numbers
- Correcting inconsistent categories
- Detecting invalid values and outliers

## Suggested Use
Students can clean the dataset using Power Query in Excel or PowerBI, Python, SQL or R.
