# CarePoint: Dirty Clinic Patient Dataset
This is a synthetic messy clinic patient dataset created for teaching data cleaning and data preparation.

## Purpose
The dataset helps data analytics students practice:
- Handling missing values
- Removing duplicates
- Standardizing date formats
- Cleaning phone numbers
- Correcting inconsistent categories
- Detecting invalid values and outliers

## Files
- messy_clinic_patients_dataset.csv
- messy_clinic_patients_dataset.xlsx

## Important Note
This dataset is fictional and does not contain real patient information.

## Suggested Use
Students can clean the dataset using Excel, Python, SQL, Power BI, or R.
